## このセクションで学ぶこと

- Claude Code を使い始めるために必要な「3 つの要素」を知る
- アカウント・アプリ・作業フォルダがそれぞれ何のためにあるのかを理解する
- 自分のパソコンですぐ用意できるものと、そうでないものを区別できる

## 用意するものは大きく 3 つだけ

新しい道具を使い始めるとき、「何をそろえればいいのか」が分からないと、最初の一歩がとても重く感じられます。Claude Code も同じです。けれども安心してください。使い始めるために必要なものは、大きく分けるとたった 3 つです。それは「アカウント」「アプリ」「作業フォルダ」です。

順番に見ていきましょう。まず **アカウント** は、サービスを使うための「あなた専用の利用者証」です。メールアドレスなどを登録して作ります。これは Claude Code に「私はこの人です」と伝え、利用できる状態にするためのものです。

次に **アプリ** は、Claude Code を実際に動かす本体です。パソコンに入れて使うソフトウェアで、これがないと相棒に話しかけることができません。スマートフォンにアプリを入れて初めて使えるようになるのと同じイメージです。

最後に **作業フォルダ** は、Claude Code に手伝ってほしいファイルをまとめて置いておく場所です。家でいうと「作業部屋」にあたります。第 1 章で見たように、Claude Code は手元のファイルに直接手を出せるのが特徴でした。その「手を出してよい範囲」を決めるのが、この作業フォルダです。

## 具体例 — 資料づくりを頼む場面で考える

たとえば、会議用の資料を Claude Code に手伝ってもらいたいとします。このとき、アカウントで「利用できる状態」になり、アプリで「話しかけられる状態」になり、資料のファイルを置いた作業フォルダで「どこを手伝ってほしいか」が決まります。3 つがそろって初めて、「この資料をもう少し読みやすく整えて」とお願いできるようになります。

3 つのうち、アカウントとアプリは最初に一度だけ用意すれば、その後はずっと使い回せます。一方、作業フォルダは、手伝ってほしい内容ごとに「今回はこのフォルダ」と選ぶ、その都度の準備だと考えてください。

## 注意点 — 一度にすべてを完璧にしなくてよい

ここで大切なのは、「最初からすべてを完璧に準備しよう」と気負わなくてよい、ということです。アカウントやアプリの細かい設定は後から見直せますし、作業フォルダも使いながら少しずつ整えていけば十分です。まずは「この 3 つが必要なんだな」という全体像をつかめれば、このセクションの目的は達成です。具体的な手順は次のセクションで順を追って見ていきます。

## まとめ

- 用意するものは「アカウント」「アプリ」「作業フォルダ」の 3 つ
- アカウントとアプリは一度用意すれば使い回せ、作業フォルダは内容ごとに選ぶ
- 最初から完璧を目指さず、まず全体像をつかめば十分
