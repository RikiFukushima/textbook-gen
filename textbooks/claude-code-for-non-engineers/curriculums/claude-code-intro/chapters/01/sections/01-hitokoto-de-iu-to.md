## このセクションで学ぶこと

- Claude Code が「自分のパソコン上で作業を手伝ってくれる AI の相棒」であること
- 言葉で頼むだけで、ファイルや資料といった実際のものに手を出してくれること
- 専門知識がなくても、ふだんの日本語で使い始められること

## Claude Code は「パソコンの中の相棒」

Claude Code(クロード・コード)を、ひとことで言うとこうなります。

> 自分のパソコン上で、お願いした作業を手伝ってくれる AI の相棒。

「AI」と聞くと、なんだか難しそうに感じるかもしれません。けれども、ここでのイメージはとてもシンプルです。あなたの隣に、頼んだことをテキパキこなしてくれる優秀なアシスタントが座っている、と思ってください。違うのは、その相棒が人ではなく、あなたのパソコンの中にいるという点だけです。

たとえば、机の上に書類が山積みになっているとします。「この資料の要点を 3 つにまとめて」とお願いすれば、相棒が中身を読んで整理してくれます。「同じような表をもう 1 枚作っておいて」と言えば、見本を見ながら作ってくれます。Claude Code は、こうした「ちょっと頼みたい作業」をパソコン上で引き受けてくれる存在です。

## 言葉で頼むだけでいい

Claude Code の大きな特徴は、**むずかしい操作を覚えなくてよい**ことです。多くのソフトは、ボタンの場所やメニューの開き方を覚えないと使えません。Claude Code は違います。やってほしいことを、ふだんの日本語で書いて伝えるだけです。

たとえば、こんな頼み方ができます。

- 「このフォルダにある文章の誤字を直して」
- 「先月の報告書をもとに、今月のひな形を作って」
- 「この一覧表を、五十音順に並べ替えて」

専門用語も、決まった言い回しも要りません。同僚にお願いするときと同じ感覚で大丈夫です。相棒はあなたの言葉を読み取って、何をすればよいかを考え、実際に作業を進めてくれます。

もし伝え方に迷っても心配は要りません。あいまいな頼み方をしても、相棒の方から「こういう意味で合っていますか?」と確認してくれることがあります。最初から正確に指示する必要はなく、対話をしながら少しずつ望むかたちに近づけていけばよい、という気軽さがあります。

## 「初めての人向けの道具」だと考えてよい

ここで大切なのは、Claude Code は**プログラミングができる人だけのものではない**ということです。名前に「Code(コード=プログラム)」と入っているので身構えてしまいがちですが、文章を書く、資料を整える、調べ物をするといった、エンジニアでない人の仕事こそ得意分野です。

この章では、まず「Claude Code とはどんな相棒なのか」という全体像をつかむことを目指します。細かい使い方や準備はあとの章でゆっくり説明しますので、ここでは「言葉で頼むと作業を手伝ってくれる、パソコンの中の相棒」という一点だけ持ち帰ってください。

なお、相棒に任せられるのは「パソコンの中でできること」が中心です。電話をかけたり、紙の書類を手で運んだりといった現実の動作はできません。あくまでパソコン上の文章やファイルを相手にする道具だと考えておくと、できること・できないことのイメージがつかみやすくなります。

## まとめ

- Claude Code は、自分のパソコン上で作業を手伝ってくれる AI の相棒です。
- むずかしい操作は不要で、ふだんの日本語で頼むだけで動いてくれます。
- 名前に反して、非エンジニアの作業こそ得意な道具です。
