## このセクションで学ぶこと

- 対話モード(REPL)を起動して終了できる
- print で文字や計算結果を表示できる
- 対話モードを電卓や動作確認にどう使うかを理解する

## 対話モードを起動する

Python を一番手軽に動かす方法が**対話モード(REPL)**です。ターミナルで `python` とだけ入力して実行すると、対話モードが始まります。

```bash
python
```

起動すると、次のような **プロンプト**(入力待ちの記号)が表示されます。

```text
Python 3.12.3 (...)
>>> 
```

この `>>>` の右側にコードを 1 行入力して Enter を押すと、その場で実行され、結果がすぐ返ってきます。書いて即結果が見られるので、ちょっとした確認に最適です。

## 具体例:計算と print

まずは計算をさせてみましょう。式を入力するだけで答えが返ります。

```python
>>> 1 + 2
3
>>> 10 * 5
50
```

文字を表示するには **print 関数**を使います。表示したい内容をかっこの中に書きます。

```python
>>> print("Hello, Python!")
Hello, Python!
>>> print(2 + 3)
5
```

文字列はクォート(`"` または `'`)で囲みます。数値の計算結果もそのまま表示できます。対話モードを電卓代わりに使ったり、「この書き方で動くかな?」を素早く試したりできるのが便利な点です。

なお対話モードでは、式を入力するだけで `print` を書かなくても結果が表示される点も覚えておくと便利です。たとえば `1 + 2` とだけ打てば `3` が返ります。これは対話モード特有の親切な動作で、後で学ぶスクリプト実行では値を表示したいときに明示的に `print` が必要になります。学習の初期はこの違いを意識せず、まずは `print` で確実に表示する習慣をつけておくと混乱しません。

## 注意点:終了の仕方

対話モードを抜けるには、`exit()` と入力するか、`Ctrl + D`(Windows では `Ctrl + Z` のあと Enter)を押します。

```python
>>> exit()
```

なお、対話モードに入力したコードは保存されません。閉じると消えてしまうため、繰り返し使いたいまとまった処理は、次のセクションで学ぶ**スクリプトファイル**に書くのが基本です。対話モードはあくまで「その場限りの実験場」と考えると役割が整理できます。

## まとめ

- `python` で対話モード(REPL)が起動し、`>>>` に書いて即実行できる。
- `print(...)` でかっこ内の値を画面に表示する。
- 対話モードの内容は保存されない。終了は `exit()`。
