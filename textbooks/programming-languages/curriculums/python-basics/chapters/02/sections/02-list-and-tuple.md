## このセクションで学ぶこと

- リストは複数の値を順序付きでまとめ、後から変更できることを理解する
- インデックスとスライスで要素を取り出し、`append` などで操作できる
- タプルは変更できないリストであり、用途に応じて使い分けられる

## リスト ― 順序付きで値をまとめる

これまでは 1 つの変数に 1 つの値を入れてきましたが、「3 人の名前」「点数の一覧」のように **複数の値をまとめて扱いたい** 場面は多くあります。そこで使うのが **リスト** です。角かっこ `[]` の中に、値をカンマで区切って並べます。

```python
fruits = ["りんご", "みかん", "ぶどう"]
scores = [80, 95, 70]
print(len(fruits))   # 3 (要素数を取得)
```

リストは異なる型を混ぜることもできますが、実務では同じ種類の値をまとめるのが一般的です。`len()` で要素の数を取得できます。

## インデックスとスライスで取り出す

各要素には先頭から `0, 1, 2, ...` という **インデックス**(位置番号)が振られています。`fruits[0]` で 1 番目の要素を取り出せます。負の数を使うと末尾から数えられ、`fruits[-1]` は最後の要素です。

```python
print(fruits[0])    # りんご
print(fruits[-1])   # ぶどう
print(fruits[0:2])  # ['りんご', 'みかん'] (スライス: 0番目から2番目の手前まで)
```

`[開始:終了]` の形で範囲を指定する書き方を **スライス** と呼びます。終了位置は「その手前まで」で、自分自身は含まない点に注意してください。

リストは作成後に中身を変えられます(ミュータブル)。末尾への追加は `append()`、削除は `remove()` を使います。

```python
fruits.append("もも")   # 末尾に追加
fruits[0] = "いちご"     # 0番目を書き換え
print(fruits)           # ['いちご', 'みかん', 'ぶどう', 'もも']
```

## タプル ― 変更できないリスト

**タプル** はリストとよく似ていますが、丸かっこ `()` で書き、**作成後に要素を変更できない**(イミュータブル)点が決定的に違います。

```python
point = (35.6, 139.7)   # 緯度・経度のように壊したくない組
print(point[0])         # 35.6
# point[0] = 0  → TypeError: 変更しようとするとエラー
```

緯度・経度や RGB の色のように「セットで意味を持ち、書き換わってほしくない値」にはタプルが向きます。変更できないぶん、うっかり書き換える事故を防げます。逆に、要素を追加・削除しながら使うデータにはリストを選びます。

注意点として、リストのコピーには癖があります。`b = a` と書くと、リストは同じ実体を 2 つの名前で共有するため、`b` への変更が `a` にも及びます。独立した複製がほしいときは `b = a.copy()` を使ってください。

## まとめ

- リストは `[]` で複数の値を順序付きでまとめ、`append` などで後から変更できる。
- インデックス(0 始まり、負で末尾)とスライス `[開始:終了]` で要素を取り出す。
- タプルは `()` で書く変更できない集まり。壊したくない組み合わせに使い分ける。
