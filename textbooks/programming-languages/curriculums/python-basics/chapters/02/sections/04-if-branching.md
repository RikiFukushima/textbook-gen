## このセクションで学ぶこと

- `if` / `elif` / `else` で条件によって処理を切り替えられる
- 比較演算子と論理演算子で条件式を組み立てられる
- インデント(字下げ)がブロックの範囲を決めることを理解する

## if 文 ― 条件で処理を分ける

プログラムは上から順に実行されますが、「点数が60以上なら合格、そうでなければ不合格」のように **条件によって処理を変えたい** ことがあります。これを **条件分岐** と呼び、`if` 文で書きます。

```python
score = 75
if score >= 60:
    print("合格")
else:
    print("不合格")
```

`if 条件:` の行末にコロン `:` を付け、条件が `True` のときに実行する処理を **インデント(字下げ)** して書きます。条件が `False` のときは `else:` 以下が実行されます。Python ではこのインデントが「ブロックの範囲」を表す文法そのものなので、半角スペース 4 つでそろえるのが慣習です。字下げを忘れたりずらしたりすると `IndentationError` になります。

## 複数の条件を分ける ― elif

3 つ以上に分けたいときは `elif`(else if の略)を使います。上から順に条件を判定し、最初に `True` になった枝だけが実行されます。

```python
score = 82
if score >= 90:
    print("優")
elif score >= 70:
    print("良")
elif score >= 60:
    print("可")
else:
    print("不可")
# → 良
```

`score` が 82 のとき、最初の `>= 90` は満たさず、次の `>= 70` で `True` になるので「良」が表示され、それ以降は評価されません。条件の順番が結果を左右する点に注意してください。

## 条件式を組み立てる演算子

条件式には **比較演算子** と論理演算子を使います。よく使うものは次のとおりです。

- 比較: `==`(等しい)、`!=`(等しくない)、`<` `<=` `>` `>=`
- 論理: `and`(かつ)、`or`(または)、`not`(否定)

```python
age = 25
if age >= 20 and age < 65:
    print("一般料金")
```

ここで注意したいのは、代入の `=` と比較の `==` の混同です。`if x = 5:` はエラーになります。「等しいか調べる」ときは必ず `==` を 2 つ重ねます。また、`if items:` のように書くと、リストが空でないとき `True`、空のとき `False` と判定でき、要素の有無チェックに便利です。

## まとめ

- `if` / `elif` / `else` で条件によって実行する処理を切り替える。
- 条件式は比較演算子(`==` `>=` など)と論理演算子(`and` `or` `not`)で組み立てる。
- インデントがブロックの範囲を決める文法。スペース 4 つでそろえる。
