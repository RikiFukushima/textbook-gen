## このセクションで学ぶこと

- 変数は値に名前を付ける箱であり、代入で値を結び付けることを理解する
- 数値(int / float)・文字列(str)・真偽値(bool)の基本型を使い分けられる
- `type()` で型を確認し、文字列と数値の変換を行える

## 変数 ― 値に名前を付ける

プログラムでは、計算結果やユーザーの入力など「あとで使いたい値」を **変数** に保存します。変数とは値に名前を付けた入れ物だと考えてください。Python では `=`(イコール)を使って値を変数に **代入** します。

```python
price = 1200       # 整数を price に代入
name = "りんご"     # 文字列を name に代入
print(name, price) # りんご 1200
```

数学の「等しい」ではなく、「右の値を左の名前に入れる」という意味である点に注意してください。`price = price + 100` のように、自分自身を使って値を更新することもできます。変数名は半角の英小文字とアンダースコアで付けるのが慣習で(例: `user_name`)、何を表す値かが分かる名前を選ぶとコードが読みやすくなります。

## 基本のデータ型

値には **データ型**(型)があり、型によってできる操作が変わります。基礎の段階で押さえるべきは次の 4 つです。

- **int(整数)**: `10` `-3` `0` など。小数を含まない数。
- **float(浮動小数点数)**: `3.14` `1.0` など。小数を含む数。
- **str(文字列)**: `"Python"` `'こんにちは'` のように引用符で囲んだ文字の並び。
- **bool(真偽値)**: `True` か `False` の 2 つだけ。条件が成り立つかどうかを表します。

`type()` を使うと、ある値や変数の型を確認できます。型が分からなくなったら確認する癖をつけましょう。

```python
print(type(10))      # <class 'int'>
print(type(3.14))    # <class 'float'>
print(type("abc"))   # <class 'str'>
print(type(True))    # <class 'bool'>
```

Python は変数を宣言するときに型を書きません。代入した値によって型が自動的に決まります(動的型付け)。そのため `x = 5` の後で `x = "five"` のように別の型を入れ直すこともできます。

## 型をまたぐときは変換する

注意したいのは、**文字列と数値は混ぜて計算できない** ことです。たとえばユーザー入力は文字列として受け取るため、そのまま足し算するとエラーになります。`int()` や `float()` で数値に、`str()` で文字列に変換します。

```python
age_text = "20"
age = int(age_text)      # 文字列 "20" を整数 20 に変換
print(age + 5)           # 25

count = 3
message = "個数: " + str(count)  # 数値を文字列に変換して連結
print(message)           # 個数: 3
```

`"20" + 5` のように型が合わないまま演算すると `TypeError` が発生します。エラーメッセージに型名(`str` や `int`)が出てきたら、まず型がそろっているかを疑ってください。なお `True` は数値の `1`、`False` は `0` として計算にも使えますが、まずは「条件の結果を表す型」として理解しておけば十分です。

## まとめ

- 変数は `=` で値に名前を付ける入れ物で、Python は代入した値から型を自動で決める。
- 基本型は int / float / str / bool の 4 つ。`type()` で確認できる。
- 文字列と数値は直接混ぜられないので、`int()` / `str()` などで変換してから扱う。
