## このセクションで学ぶこと

- `open()` でファイルを開き、モード(`r` / `w` / `a`)を使い分けられる
- `read` / `readline` / `readlines` や `for` 文でファイルの中身を読める
- `write` でファイルに文字列を書き出し、最後に `close` する流れを理解する

## ファイルを開く ― open() とモード

プログラムが扱うデータの多くは、テキストファイルや CSV などの「ファイル」としてディスクに保存されています。Python ではファイルを **開く → 読み書きする → 閉じる** という 3 ステップで操作します。最初のステップが組み込み関数 `open()` です。

`open()` の第 1 引数にはファイルのパス、第 2 引数には **モード** を渡します。モードは「そのファイルを何のために開くか」を表す文字で、主に次の 3 つを使い分けます。

| モード | 意味 | 既存ファイルがあると |
| --- | --- | --- |
| `"r"` | 読み込み(read) | 中身をそのまま読む |
| `"w"` | 書き込み(write) | 中身を消して上書きする |
| `"a"` | 追記(append) | 末尾に書き足す |

モードを省略すると `"r"`(読み込み)になります。日本語などのテキストを扱うときは、文字コードのずれを防ぐため `encoding="utf-8"` を添えるのが実務での基本です。

```python
f = open("memo.txt", "r", encoding="utf-8")
```

## ファイルを読む

開いたファイルオブジェクトからは、いくつかの方法で中身を取り出せます。`read()` はファイル全体を 1 つの文字列として返します。1 行ずつ読みたいときは、ファイルオブジェクトを `for` 文に渡すのがもっとも簡潔で実用的です。

```python
f = open("memo.txt", "r", encoding="utf-8")

# 方法 1: 全体をまとめて読む
text = f.read()
print(text)

f.close()

# 方法 2: 1 行ずつ読む(行末の改行はそのまま含まれる)
f = open("memo.txt", "r", encoding="utf-8")
for line in f:
    print(line.rstrip())  # rstrip() で末尾の改行を取り除く
f.close()
```

このほか、1 行だけ取り出す `readline()`、全行をリストにして返す `readlines()` もあります。巨大なファイルを `read()` で一気に読むとメモリを圧迫するため、行単位で処理できる `for` 文のほうが安全です。

## ファイルに書く

書き込みは `write()` メソッドで行います。モード `"w"` で開くと **既存の中身は消えて上書き** される点に注意してください。消したくない場合は追記モード `"a"` を使います。`write()` は `print()` と違って自動で改行を付けないため、改行したいときは `\n` を自分で書きます。

```python
f = open("log.txt", "w", encoding="utf-8")
f.write("1 行目\n")
f.write("2 行目\n")
f.close()

# 追記モードなら既存内容の末尾に足される
f = open("log.txt", "a", encoding="utf-8")
f.write("3 行目を追加\n")
f.close()
```

## 注意点

- 開いたファイルは必ず `close()` で閉じます。閉じ忘れると書き込みが確定しなかったり、資源が解放されなかったりします。
- `"w"` モードは中身を消すため、大切なファイルを誤って空にしないよう、パスとモードをよく確認してください。
- 日本語が文字化けするときは、ほぼ `encoding` の指定漏れが原因です。読み書きの両方で `encoding="utf-8"` をそろえましょう。

なお、`close()` の呼び忘れを防ぐより安全な書き方として、次のセクションで学ぶ `with` 文があります。実務では `with` 文を使うのが基本になります。

## まとめ

- ファイルは `open()` で開き、モード `r` / `w` / `a` で読み込み・上書き・追記を切り替える。
- 読み込みは `read()` や `for line in f` で、書き込みは `write()` で行う。
- 開いたファイルは必ず `close()` で閉じる(次節の `with` 文ならより安全)。
