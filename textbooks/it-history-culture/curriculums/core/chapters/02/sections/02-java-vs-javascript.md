## このセクションで学ぶこと

- Java の名前がコーヒーに由来し、最初は「Oak」という別名だったこと
- JavaScript がわずか 10 日で試作され、マーケティング上の理由で Java の名を借りたこと
- Java と JavaScript は名前が似ているだけの、まったく別の言語であること

## 樫の木からコーヒーへ — Java 誕生

1991 年、米 Sun Microsystems で「Green Project」という秘密プロジェクトが始まりました。ジェームズ・ゴスリングらが目指したのは、家電製品向けの新しいプログラミング言語。ゴスリングはこの言語を Oak(樫)と名付けます。由来は、オフィスの窓の外に立っていた 1 本の樫の木でした。

ところが世に出す段になって問題が発覚します。Oak はすでに別の会社が商標として使っていて、その名前では発表できないのです。改名のためのブレインストーミングを重ねた末、1995 年に選ばれた名前が Java。アメリカで「java」はコーヒーを指す俗語で、その語源はコーヒーの名産地、インドネシアのジャワ島です。Java のロゴが湯気の立つコーヒーカップなのは、この由来をそのまま絵にしたものです。Java は「Write Once, Run Anywhere(一度書けば、どこでも動く)」を合言葉に 1995 年 5 月にデビューし、IT 業界最大の話題になりました。

## 10 日間で作られた言語 — JavaScript 誕生

同じ 1995 年、ブラウザの会社 Netscape では、ブレンダン・アイクという技術者が「ブラウザの中で動く手軽な言語」をわずか 10 日で試作していました。開発コード名は Mocha(モカ)、ベータ版公開時の名前は LiveScript。ここまで、Java とは何の関係もありません。

転機は 1995 年 12 月。Netscape は Sun との提携を発表し、それに合わせて言語名を JavaScript に変更します。当時 Java は業界で最も注目を集める言語であり、その人気にあやかるための、純粋にマーケティング上の命名でした。つまり JavaScript は、生まれも設計者も中身も Java とは別物なのに、名前だけを借りた言語なのです。

## 「メロンとメロンパン」

このため両者の関係は、よく「メロンとメロンパンくらい違う」とたとえられます(英語圏では「car と carpet」)。文法の見た目はどちらも C 系の言語として似ていますが、設計思想も得意分野も別。Java は大規模な業務システムや Android アプリで、JavaScript は Web ブラウザの中で育ちました。

名前の貸し借りには後日談もあります。「JavaScript」という商標は Sun(後に同社を買収した Oracle)が保有し続けたため、言語を国際標準にする際にこの名前が使えず、標準規格としての正式名称は ECMAScript(エクマスクリプト)という別名になりました(1997 年に標準化)。世界で最も使われる言語のひとつが、自分の名前を自由に使えない — 命名の経緯が残した、少し不思議なねじれです。

## 注意点 — 「JavaScript は Java の簡易版」ではない

「JavaScript は Java の簡易版・Web 版」という説明をときどき見かけますが、これは誤りです。歴史的に見れば「名前を借りた」だけで、技術的な親子関係はありません。混同して書かれた求人票や記事は今でも珍しくないので、「名前が似ているだけの別言語」とだけ覚えておけば十分です。

## まとめ

- Java は Oak(樫の木)→ 商標問題 → コーヒーの俗語 Java、という経緯で命名された
- JavaScript は Netscape が 10 日で試作した言語で、Java 人気にあやかって改名された
- 両者は名前が似ているだけの別言語。標準規格名が ECMAScript なのも商標問題の名残
