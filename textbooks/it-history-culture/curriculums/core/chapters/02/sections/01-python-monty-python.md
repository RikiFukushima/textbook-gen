## このセクションで学ぶこと

- Python という名前がヘビではなく、英国のコメディ番組に由来すること
- Python が 1989 年のクリスマス休暇の「暇つぶし」から生まれた経緯
- spam と eggs、BDFL など、いまも Python 文化に残る遊び心の痕跡

## クリスマス休暇の暇つぶしから

1989 年 12 月、オランダ・アムステルダム。国立の研究機関 CWI に勤めるプログラマ、グイド・ヴァンロッサムは、ちょっとした問題を抱えていました。職場はクリスマス休暇で閉まっているのに、自宅にはこれといってやることがない。そこで彼は「前から構想していた新しいプログラミング言語でも書いてみるか」と、自宅のコンピュータに向かいます。

いまや世界で最も使われる言語のひとつが、休暇の暇つぶしとして産声を上げた瞬間でした。グイドは以前、初心者向けの教育用言語 ABC の開発に関わっており、その「読みやすさを最優先する」という思想を新しい言語に受け継がせます。こうして書き始められた言語は 1991 年 2 月に最初の版が公開され、少しずつ世界へ広がっていきました。

## 名前の由来はコメディ番組

新しい言語には名前が必要です。グイドが求めた条件は「短くて、少し謎めいていて、覚えやすいこと」。そのとき彼が愛読していたのが、英国 BBC のコメディ番組「空飛ぶモンティ・パイソン(Monty Python's Flying Circus)」の台本集でした。彼はこの番組の大ファンだったのです。こうして言語の名前は Python に決まりました。つまり Python は、ヘビの名前ではなくコメディ番組の名前なのです。

とはいえ、世間は素直に「Python = ニシキヘビ」と受け取りました。公式ロゴは 2 匹のヘビを組み合わせたデザインになり、解説書の表紙にもしばしばヘビが描かれます。グイド本人もこの「ヘビ解釈」を否定せず、むしろ楽しんで受け入れています。

## いまも残るコメディの痕跡

プログラミングの解説では、仮の名前として foo や bar を使う伝統があります。ところが Python の公式チュートリアルでは、代わりに spam と eggs が登場します。これはモンティ・パイソンの有名なコント「スパム」— 何を注文しても缶詰肉のスパムが付いてくる食堂の話 — へのオマージュです。ちなみに、迷惑メールを「スパム」と呼ぶ語源も同じコントだとされています。

グイドはその後も長く、「優しい終身の独裁者(BDFL: Benevolent Dictator For Life)」というユーモラスな肩書きで Python の最終決定権を持ち続けました(2018 年に自ら退任)。冗談のような肩書きを公式に名乗り続けるところにも、この言語コミュニティの遊び心が表れています。

## 注意点 — 遊びから生まれても、設計は本気

誕生の経緯も名前も軽やかですが、Python の設計思想は一貫して真剣です。「読みやすいコードこそ正義」という価値観は 30 年以上守られ続け、その読みやすさゆえに教育現場で採用され、やがてデータ分析や AI 開発の標準言語にまで上り詰めました。「クリスマスの暇つぶし」と「AI 時代の主役」が同じ言語だという落差こそ、この物語のいちばんの面白さです。

## まとめ

- Python の名前は英国のコメディ番組「空飛ぶモンティ・パイソン」に由来し、ヘビは後付けの解釈
- 1989 年のクリスマス休暇に、グイド・ヴァンロッサムが暇つぶしとして書き始めた
- spam と eggs、BDFL という肩書きなど、遊び心はいまも Python 文化に残っている
