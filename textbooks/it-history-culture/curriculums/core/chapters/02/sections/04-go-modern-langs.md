## このセクションで学ぶこと

- Go が Google 社内の「コンパイル待ち 45 分」への不満から生まれたこと
- B を作ったケン・トンプソンが、約 40 年後に Go の共同設計者として再登場すること
- Rust や Kotlin など、現代の言語も企業の現実的な課題から生まれていること

## コンパイル待ちの雑談から

2007 年、Google。同社の巨大な C++ プログラムは、コンパイル(人間が書いたコードを機械が実行できる形に変換する処理)に 45 分もかかることがありました。その待ち時間に雑談をしていた 3 人のエンジニア — ロバート・グリーズマー、ロブ・パイク、ケン・トンプソン — は、「待たされるくらいなら、速くてシンプルな言語を自分たちで設計しよう」と話を始めます。これが Go の出発点です。Go は 2009 年 11 月に公開され、2012 年に正式版 1.0 が登場しました。

ここで 3 人目の名前に注目してください。ケン・トンプソン — 前のセクションに登場した、1969 年に言語 B を作り、Unix を生んだ本人です。B から C が生まれたのが 1972 年。その張本人が、約 40 年後に C++ への不満から新しい言語を設計している。言語の歴史が一周して戻ってきたような、できすぎた巡り合わせです。

## 短すぎる名前の悩み

Go という名前は「短く、シンプルに」という設計思想そのものですが、短すぎて困ったことが起きました。「go」はあまりに一般的な英単語で、検索してもまるで絞り込めないのです。そこで公式サイトのドメイン golang.org にちなんだ「Golang(ゴーラング)」という通称が定着しました。マスコットはホリネズミの Gopher(ゴーファー)。ゆるい見た目で世界中の開発者に愛されています。

## 各社の事情、各様の言語

同じ時代の言語たちも、それぞれの企業の「困りごと」から生まれています。

- **Rust**(2015 年に 1.0): ブラウザ開発企業の Mozilla が支援して育てた言語。C++ で書かれたブラウザはメモリ管理のミスが安全上の弱点の温床で、それを言語の仕組みごと防ぐために生まれました。名前について作者のグレイドン・ホアは「さび菌(植物に寄生する、しぶとい菌類)から取った」と説明しています。
- **Kotlin**(2011 年発表): 開発ツール企業 JetBrains 製。名前はロシア・サンクトペテルブルク沖のコトリン島から。「Java が島の名前なら、こちらも島の名前で」という Java への目配せです。2017 年には Android 開発の公式言語に採用されました。
- ほかにも Apple の Swift(2014 年)、Microsoft の TypeScript(2012 年)など、現代の主要言語の多くは企業発です。

## 注意点 — 企業発でも「閉じた言語」ではない

企業の都合から生まれた言語と聞くと、その会社専用の閉じた道具を想像するかもしれません。実際は逆で、Go も Rust も Kotlin もオープンソースとして公開され、社外の開発者を巻き込みながら育っています。「企業の課題から生まれ、公開されてみんなの道具になる」— この流れを支える文化については、第 4 章のオープンソースの物語で詳しく見ていきます。

## まとめ

- Go は Google 社内の C++ コンパイル待ちへの不満から生まれ、通称 Golang と呼ばれる
- 共同設計者ケン・トンプソンは B と Unix の生みの親で、言語史が一周した巡り合わせ
- Rust(さび菌)や Kotlin(コトリン島)など、現代の言語も企業の課題から生まれ、オープンソースで育っている
