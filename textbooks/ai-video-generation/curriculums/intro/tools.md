# tools.md ― ツール名・価格・モデル世代の差し替え表

本文(各セクションの Markdown)では、ツール名・価格・モデル世代を **直接書かず** に
`{{tool:…}}` / `{{price:…}}` / `{{spec:…}}` / `{{model:…}}` というプレースホルダで書いています。
実際の名称・数値はすべてこのファイルに集約します。**陳腐化したらこのファイルだけを更新**してください
(本文には手を入れない)。

> ⚠️ 下表の「現在の実体」は記入例です。AI動画生成ツールは更新が速いため、
> **利用前に必ず最新情報を確認して差し替えてください**。価格・無料枠・モデル世代は変わります。

- 最終確認日: （記入してください）

## プレースホルダ一覧

本文に登場するキーは下表のものだけです。新しいキーを増やすときは、本文に書く前に必ずここへ追記してください。

### ツール(`{{tool:…}}`)

| プレースホルダ | 意味(本文での役割) | 現在の実体(記入例 / 要差し替え) |
| --- | --- | --- |
| `{{tool:動画生成}}` | テキストや画像から動画を生成する主役ツール | （例: 代表的なテキスト/画像→動画ツール） |
| `{{tool:画像から動画}}` | 用意した1枚の画像を動かす image-to-video のツール/機能 | （例: image-to-video 対応ツール） |
| `{{tool:画像生成}}` | 元画像を用意するための画像生成ツール | （例: 代表的な画像生成ツール） |
| `{{tool:仕上げ}}` | カット・連結・テロップ・書き出しができる軽い編集ツール | （例: スマホ/ブラウザで使える無料の編集アプリ） |

### 価格・課金(`{{price:…}}`)

## 運用メモ

- 本文側にプレースホルダ以外のツール名・価格・モデル名を書かない。一般的な概念(「クレジット制」「透かし」「アスペクト比」など)は普通の言葉でよい。
- 表示用にプレースホルダを実名へ置換するビルド手順は今後の課題。現状はこの表を見ながら読み替える。
